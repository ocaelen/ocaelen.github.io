# Olivier Caelen personal webpage

This is a simple static website for GitHub Pages.

## Deploying on GitHub Pages

Use the repository root as the GitHub Pages publishing source. The homepage is `index.html`.

The `.nojekyll` file disables Jekyll processing, which keeps GitHub Pages from trying to build this plain HTML/CSS site.

## Updating the site

- Edit `index.html` for content changes.
- Edit `styles.css` for visual changes.
- When adding a publication at the top of the list, update the `start` value on the publication `<ol>` to the new total number.
- Keep private source files, such as Word documents, out of the deployed repository.
